Sorting and Searching

The Algorithm Design Manual Chap4. Sorting and Searching

最优排序时间：O(nlog(n))。
排序的应用：（做算法题时可参考）
An important algorithm design technique is to use sorting as a basic building block, because many other problems become easy once a set of items is sorted.

  1. 查找某数字——将数组排序，此情况下查找可做到O(log(n))。查找是排序最重要的应用。
  2. Closest pair（差值最小的数对）——将数组排序后遍历即可，只需要linear time。
  3. 元素重复性（是否存在重复元素）——2的特例，排序后进行linear scan，找到差值为0的数对。
  4. 找出元素k重复的次数——将数组排序，找到k-ϵ和k+ϵ的序号并相减（ϵ为任意足够小的数），时间为O(log(n))。
  5. 选择第k大的数——排序后，constant time查找第k个数即可。特殊：第n/2个数是中位数。
  6. Convex hulls（凸包；凸壳）——没看懂，原书P105
Sorting lies at the heart of many algorithms. Sorting the data is one of the first things any algorithm designer should try in the quest for efficiency.
